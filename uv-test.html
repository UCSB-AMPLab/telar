UV Baseline Test

Same 6 tests as the Tify evaluation, using UniversalViewer v4.0.0 for comparison.

Test 1: Basic Loading & OSD Access

Load UV with the local leviathan manifest. Poll for OSD via _assignedContentHandler.

Test 2: Programmatic Viewport Control

Test panTo, zoomTo, getHomeZoom, getHomeBounds — the methods Telar's story engine uses.

Test 3: External IIIF Manifest

Load a remote Presentation API v2 manifest from Princeton (Figgy).

Test 4: Chrome Hiding (Story Mode)

Same manifest, but with all UV chrome hidden — only the OSD canvas should be visible.

Test 5: Multiple Concurrent Instances

Create 10 simultaneous UV instances. Measure init time, memory, and DOM nodes.

Test 6: Init Timing Benchmark

Create and time a single UV instance from constructor to OSD-ready. Run 5 times.
