Tify Viewer Evaluation

Research spike for Telar v0.8.2 — testing Tify as a potential UniversalViewer replacement.

Test 1: Basic Loading & OSD Access

Load Tify with the local leviathan manifest. Verify tify.viewer exposes OSD.

Test 2: Programmatic Viewport Control

Test panTo, zoomTo, getHomeZoom, getHomeBounds — the methods Telar's story engine uses.

Test 3: External IIIF Manifest

Load a remote Presentation API v2 manifest from Princeton (Figgy).

Test 4: Chrome Hiding (Story Mode)

Same manifest, but with all Tify UI hidden — only the OSD canvas should be visible.

Test 5: Multiple Concurrent Instances

Create 10 simultaneous Tify instances. Measure init time, memory, and DOM nodes.

Test 6: Init Timing Benchmark

Create and time a single Tify instance from constructor to OSD-ready. Run 5 times.
